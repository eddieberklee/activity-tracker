== Activity Tracker

An app that allows you to track which activities you want to work on daiy.

The goal is for you to be able to track of which activities you have done today and which activities you want to accomplish.

We all have skills we should be honing on a daily basis but sometimes we forget or lose sight of how much of a difference doing something, a little bit each day, can have in the long-term.
